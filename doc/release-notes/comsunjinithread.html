com.sun.jini.thread package
v2.1.2 Release Notes

    Description

    The com.sun.jini.thread package contains various thread-related utility classes and interfaces. These classes and interfaces are not intended for general use, but are used internally by other parts of the Apache River release. Some of these classes and interfaces are exposed via the configuration mechanism.

    Changes since the v2.1.1 release
