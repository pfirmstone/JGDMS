Jini(TM) Network Technology Specifications
Apache River v2.2.1

    The following specifications are available in HTML in this v2.2.1 release.

    The existing specifications were accepted as standards of the Jini Community(SM) through the Jini Community Decision Process (JDP) and have been inherited by the Apache River project. Modifications are to be expected based on learnings from Apache River's continued development.
