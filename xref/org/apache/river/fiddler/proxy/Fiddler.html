36  /**
37   * This interface defines the private protocol between the client-side
38   * proxy and the server of a lookup discovery service.
39   * <p>
40   * The declared methods mirror the methods of the LookupDiscoveryService 
41   * interface. Classes that act as a "smart proxy" to the lookup discovery
42   * service typically contain a field which implements this interface. 
43   * When the lookup discovery service registers with a lookup service,
44   * the proxy - containing a "server field" that is an instance of this
45   * interface - is registered. It is through the server field that the
46   * proxy interacts with the back-end server of the lookup discovery service,
47   * on behalf of a client, using the private protocol defined by a class
48   * that implements this interface.
49   *
50   * @author Sun Microsystems, Inc.
51   *
52   */
53  public interface Fiddler extends Remote, Administrable, FiddlerAdmin,
54                            ServiceProxyAccessor
55  {
56      /**
57       * Returns the unique identifier generated (or recovered) by the back-end
58       * implementation of the lookup discovery service when an instance of
59       * that service is constructed. This ID is typically used to determine
60       * equality between the proxies of any two instances of the lookup
61       * discovery service.
62       * 
63       * @return the unique ID that was generated (or recovered) by the
64       *         back-end implementation of the lookup discovery service
65       *         at creation time
66       * 
67       * @throws java.rmi.RemoteException typically, this exception occurs when
68       *         there is a communication failure between the client and the
69       *         lookup discovery service.
70       */
71      Uuid getProxyID() throws RemoteException;

79       *                      consisting of zero or more names of groups to
80       *                      which lookup services to discover belong.
81       *                      
82       * @param locators      array of zero or more non-null LookupLocator
83       *                      objects, each corresponding to a specific lookup
84       *                      service to discover.
85       *                      
86       * @param listener      a non-null instance of RemoteEventListener. This 
87       *                      argument specifies the entity that will receive
88       *                      events notifying the registration that a lookup
89       *                      service of interest has been discovered. 
90       *                      
91       * @param handback      null or an instance of MarshalledObject. This
92       *                      argument specifies an object that will be 
93       *                      included in the notification event that the
94       *                      lookup discovery service sends to the registered
95       *                      listener.
96       *                      
97       * @param leaseDuration long value representing the amount of time (in
98       *                      milliseconds) for which the resources of the
99       *                      lookup discovery service are being requested.
100      *                      
101      * @return an instance of the LookupDiscoveryRegistration interface.
102      * 
103      * @throws java.rmi.RemoteException typically, this exception occurs when
104      *         there is a communication failure between the client and the
105      *         lookup discovery service. When this exception does occur, the
106      *         registration may or may not have completed successfully.

117     /**
118      * Returns an array consisting of instances of the ServiceRegistrar
119      * interface. Each element in the returned set is a proxy to one of
120      * the lookup service(s) that have already been discovered for the
121      * registration corresponding to the <code>registrationID</code> 
122      * parameter. Each element of the return set is a marshalled instance
123      * of the <code>ServiceRegistrar</code> interface. The contents of the
124      * returned set make up the 'remote state' of the registration's
125      * currently discovered lookup service(s).
126      *
127      * @param registrationID unique identifier assigned to the registration
128      *                       from which the set of registrars is being 
129      *                       retrieved
130      * 
131      * @return an array of MarshalledObject objects where each element is
132      *         is a marshalled instance of ServiceRegistrar.
133      * 
134      * @throws java.rmi.RemoteException typically, this exception occurs when
135      *         there is a communication failure between the client and the
136      *         lookup discovery service.
137      *
138      * @throws org.apache.river.proxy.ThrowThis which is a non-remote "wrapper"
139      *         class used to wrap various remote exceptions (for example,
140      *         NoSuchObjectException) that this method wishes to throw.
141      *         When a service is implemented as a smart proxy with a
142      *         backend server, and a method on the backend which was invoked
143      *         through the proxy wishes to explicitly throw a particular
144      *         remote exception, it cannot simply throw that exception if
145      *         it wishes that exception to be visible to the proxy running
146      *         on the "client side". This is because when the backend throws
147      *         any remote exception, the RMI sub-system automatically wraps
148      *         that exception in a java.rmi.ServerException. Thus, the proxy
149      *         will only be able to "see" the ServerException (the actual
150      *         exception that the backend tried to throw is "buried" in the
151      *         detail field of the ServerException). Thus, in order to allow
152      *         the proxy access to the actual remote exception this method
153      *         throws, that exception wraps the desired remote exception in
154      *         the non-remote exception ThrowThis; which will not be wrapped
155      *         in a ServerException.
156      *
157      *         This method throws a NoSuchObjectException wrapped in a
158      *         ThrowThis exception whenever the <code>registrationID</code>
159      *         parameter references an invalid or non-existent registration.
160      *
161      * @see net.jini.discovery.LookupDiscoveryRegistration#getRegistrars
162      */
163     MarshalledObject[] getRegistrars(Uuid registrationID)
164                                              throws RemoteException, ThrowThis;
