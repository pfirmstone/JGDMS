43  /**
44   * Defines a {@link DynamicPolicy} that logs information about missing
45   * permissions, and optionally grants all permissions, which is <b>FOR
46   * DEBUGGING ONLY</b>. Do not use this security policy provider to grant
47   * all permissions in a production environment. <p>
48   *
49   * This class is intended to simplify the process of deciding what security
50   * permissions to grant to run an application.  While it is generally
51   * acceptable to grant all permissions to local, trusted code, downloaded
52   * code should typically be granted the least permission possible. <p>
53   *
54   * The usual approach to choosing which permissions to grant is to start by
55   * running the application with a security policy file that grants all
56   * permissions to local, trusted code.  When the application fails with an
57   * exception message that identifies a missing permission, add that
58   * permission to the security policy file, and repeat the process. Although
59   * straight forward, this process can be time consuming if the application
60   * requires many permission grants. <p>
61   *
62   * Another approach is to set the value of the
63   * <code>"java.security.debug"</code> system property to
64   * <code>"access,failure"</code>, which produces debugging output that
65   * describes permission grants and failures. Unfortunately, this approach
66   * produces voluminous output, making it difficult to determine which
67   * permission grants are needed. <p>
68   *
69   * This security policy provider permits another, hopefully more
70   * convenient, approach. When this class is specified as the security
71   * policy provider, and granting all permissions is enabled, it uses the
72   * standard dynamic security policy to determine what permissions are
73   * granted. If a permission is not granted by the standard policy, though,
74   * then rather than denying permission, this class logs the missing
75   * permission in the form required by the security policy file, and grants
76   * the permission, allowing the program to continue. In this way,
77   * developers can determine the complete set of security permissions
78   * required by the application. <p>
79   *
80   * Note that the information printed by this security policy provider may
81   * not be in the form you wish to use in your policy file. In particular,
82   * using system property substitutions and <code>KeyStore</code> aliases
83   * may produce a more portable file than one containing the exact entries
84   * logged. Note, too, that the information printed for
85   * <code>signedBy</code> fields specifies the principal name for
86   * <code>X.509</code> certificates, rather than the <code>KeyStore</code>
87   * alias, which is not a valid security policy file format. <p>
88   *
89   * Using this security policy provider without granting all permissions is
90   * also useful since it prints information about security exceptions that
91   * were caught, but that might have an affect on program behavior. <p>
92   *
93   * This class uses uses the {@link Logger} named
94   * <code>net.jini.security.policy</code> to log information at the following
95   * levels: <ul>
96   *
97   * <li> {@link Level#WARNING WARNING} - Permissions that were needed but not
98   * granted by the policy file.
99   *
100  * <li> {@link Level#FINE FINE} - Also include stack traces.
101  *
102  * <li> {@link Level#FINER FINER} - All permissions granted, with stack traces
103  * for ones not granted by the policy file, and dynamic grants.
104  *
105  * <li> {@link Level#FINEST FINEST} - All permissions granted, with all stack
106  * traces, and dynamic grants. </ul>
107  *
108  * To use this security policy provider, do the following: <ul>
109  *
110  * <li> Copy the <code>jsk-policy.jar</code> file from the <code>lib-ext</code>
111  * subdirectory of the JGDMS release
112  * installation to the extensions directory of the Java(TM) 2 SDK (or JRE)
113  * installation, and copy the <code>jsk-debug-policy.jar</code> file
114  * from the <code>lib</code> subdirectory of the JGDMS release installation to
115  * the extensions directory of the Java 2 SDK (or JRE) installation.
116  *
117  * <li> Specify this class as the security policy provider. Create a copy of
118  * the file <code>jre/lib/security/security/java.security</code>, modify the
119  * file to contain the line:
120  *
121  * <blockquote>
122  * <pre>
123  * policy.provider=org.apache.river.tool.DebugDynamicPolicyProvider
124  * </pre>
125  * </blockquote>
126  *
127  * and then specify this new file as the value of the
128  * <code>java.security.properties</code> system property.  
129  *
130  * <li> Specify whether all permissions should be granted by setting the
131  * <code>org.apache.river.tool.DebugDynamicPolicyProvider.grantAll</code> security
132  * property to <code>true</code> by adding the following line to the security
133  * properties file:
134  * 
135  * <blockquote>
136  * <pre>
137  * org.apache.river.tool.DebugDynamicPolicyProvider.grantAll=true
138  * </pre>
139  * </blockquote> </ul> <p>
140  *
141  * Granting all permissions is disabled by default. <p>
142  *
143  * Make sure to specify a security manager, either by setting the
144  * <code>java.security.manager</code> system property, or putting the following
145  * code in the main method of the application:
146  *
147  * <blockquote>
148  * <pre>
149  * if (System.getSecurityManager() == null) {
150  *     System.setSecurityManager(new SecurityManager());
151  * }
152  * </pre>
153  * </blockquote>
154  *
155  * <p>This provider can be used in conjunction with the provider
156  * <code>org.apache.river.start.AggregatePolicyProvider</code> by setting the
157  * <code>org.apache.river.start.AggregatePolicyProvider.mainPolicyClass</code> 
158  * system property to the fully qualified name of this class.  If this
159  * provider is used with the <code>AggregatePolicyProvider</code>, then the
160  * JAR file <code>jsk-debug-policy.jar</code> needs to be in the
161  * application's class path, and this class needs to be granted all
162  * permissions. 
163  *
164  *
165  * @author Sun Microsystems, Inc.
166  **/
167 public class DebugDynamicPolicyProvider extends DynamicPolicyProvider {

169     /* Logger to use */
170     private static final Logger logger =
171 	Logger.getLogger("net.jini.security.policy");

173     /* If true, always grant permission */
174     private static boolean grantAll =
175 	((Boolean) AccessController.doPrivileged(
176 	    new PrivilegedAction() {
177 		public Object run() {
178 		    return Boolean.valueOf(
179 			Security.getProperty(
180 			    "org.apache.river.tool." +
181 			    "DebugDynamicPolicyProvider.grantAll"));
182 		}
183 	    })).booleanValue();

185     /* Cache of permission requests already made */
186     private static final Set requests = new HashSet();
187 
188     /** The empty codesource. */
189     private static final CodeSource emptyCS =
190 	new CodeSource(null, (Certificate[]) null);

192     /**
193      * Creates an instance of this class that wraps a default underlying
194      * policy, as specified by {@link
195      * DynamicPolicyProvider#DynamicPolicyProvider() DynamicPolicyProvider()}.

197      * @throws PolicyInitializationException if unable to construct the base
198      *	       policy
199      * @throws SecurityException if there is a security manager and the calling
200      *	       context does not have adequate permissions to read the <code>
201      *	       net.jini.security.policy.DynamicPolicyProvider.basePolicyClass
202      *	       </code> security property, or if the calling context does not
203      *	       have adequate permissions to access the base policy class

205     public DebugDynamicPolicyProvider() throws PolicyInitializationException { }

208      * Creates an instance of this class that wraps around the given

212      *        non-dynamic grants

214      * 	       <code>null</code>

216     public DebugDynamicPolicyProvider(Policy basePolicy) {
217 	super(basePolicy);

220     /** Log calls. */
221     public void grant(Class cl,
222 		      Principal[] principals,
223 		      Permission[] permissions)
224     {
225 	try {
226 	    super.grant(cl, principals, permissions);
227 	    if (permissions == null
228 		|| permissions.length == 0
229 		|| !logger.isLoggable(Level.FINER))
230 	    {
231 		return;
232 	    }
233 	    Request req = new Request(cl, principals, permissions);
234 	    if (cl == null) {
235 		logger.log(Level.FINER,
236 			   "Granting permissions for all classes:\n{0}",
237 			   req.toString());
238 	    } else {
239 		logger.log(Level.FINER,
240 			   "Granting permissions for {0}:\n{1}",
241 			   new Object[] { cl, req.toString() });
242 	    }
243 	} catch (SecurityException e) {
244 	    if (logger.isLoggable(Level.FINE)) {
245 		logger.log(Level.FINE, "Granting permissions failed", e);
246 	    }
247 	    throw e;
248 	}
249     }
250 
251     /** Always returns true, but logs unique requests */
252     public boolean implies(ProtectionDomain pd, Permission perm) {
253 	boolean implies = super.implies(pd, perm);
254 	boolean result = implies ? true : grantAll;
255 	if (!(logger.isLoggable(Level.FINER)
256 	      || (!implies && logger.isLoggable(Level.WARNING))))
257 	{
258 	    return result;
259 	}
260 	Request request = new Request(pd, perm);
261 	synchronized (requests) {
262 	    if (requests.contains(request)) {
263 		return result;
264 	    }
265 	    requests.add(request);
266 	}
267 	String stackTrace = null;
268 	if (logger.isLoggable(Level.FINEST)
269 	    || (!implies && logger.isLoggable(Level.FINE)))
270 	{
271 	    StringWriter sw = new StringWriter();
272 	    PrintWriter pw = new PrintWriter(sw);
273 	    new Exception("Stack trace:").printStackTrace(pw);
274 	    pw.close();
275 	    stackTrace = sw.toString();
276 	}
277 	if (implies) {
278 	    if (logger.isLoggable(Level.FINEST)) {
279 		logger.log(Level.FINEST, "Permission granted:\n{0}\n{1}",
280 			   new Object[] { request.toString(), stackTrace });
281 	    } else {
282 		logger.log(Level.FINER, "Permission granted:\n{0}",
283 			   request.toString());
284 	    }
285 	} else if (logger.isLoggable(Level.FINE)) {
286 	    logger.log(Level.WARNING,
287 		       (grantAll
288 			? "Permission not granted by base policy:\n{0}\n{1}"
289 			: "Permission not granted:\n{0}\n{1}"),
290 		       new Object[] { request.toString(), stackTrace });
291 	} else {
292 	    logger.log(Level.WARNING,
293 		       (grantAll
294 			? "Permission not granted by base policy:\n{0}"
295 			: "Permission not granted:\n{0}"),
296 		       request.toString());
297 	}   
298 	return result;
299     }

301     /** Returns the name of the certificate. */
302     private static String getCertName(Certificate cert) {
303 	if (cert instanceof X509Certificate) {
304 	    return ((X509Certificate) cert).getSubjectDN().getName();
305 	} else {
306 	    return cert.toString();
307 	}
308     }

310     /**
311      * Returns a quoted version of the argument, such that it would result in
312      * the argument if read from a file with the standard String syntax.
313      */
314     private static String quoteString(String s) {
315 	if (s == null) {
316 	    return "";
317 	}
318 	int len = s.length();
319 	StringBuffer buf = new StringBuffer(len + 2);
320 	buf.append('"');
321 	for (int off = 0; off < len; ) {
322 	    int quote = s.indexOf('"', off);
323 	    int slash = s.indexOf('\\', off);
324 	    if (quote >= 0 && (slash < 0 || slash > quote)) {
325 		buf.append(s.substring(off, quote));
326 		buf.append("\\\"");
327 		off = quote + 1;
328 	    } else if (slash >= 0) {
329 		buf.append(s.substring(off, slash));
330 		buf.append("\\\\");
331 		off = slash + 1;
332 	    } else {
333 		buf.append(s.substring(off));
334 		break;
335 	    }
336 	}
337 	buf.append('"');
338 	return buf.toString();
339     }

341     /* Keeps track of an individual permission request */
342     private static class Request {
343 	final CodeSource codeSource;
344 	final Certificate[] certs;
345 	final Principal[] principals;
346 	final Permission[] perms;
347 
348 	Request(ProtectionDomain pd, Permission perm) {
349 	    codeSource = pd.getCodeSource();
350 	    certs = codeSource == null ? null : codeSource.getCertificates();
351 	    principals = pd.getPrincipals();
352 	    this.perms = new Permission[] { perm };
353 	}
354 
355 	Request(final Class cl, Principal[] principals, Permission[] perms) {
356 	    codeSource = (cl == null) ? emptyCS :
357 		(CodeSource) AccessController.doPrivileged(
358 		    new PrivilegedAction() {
359 			public Object run() {
360 			    return cl.getProtectionDomain().getCodeSource();
361 			}
362 		    });
363 	    certs = null;
364 	    this.principals = principals;
365 	    this.perms = perms;
366 	}
367 
368 	public boolean equals(Object o) {
369 	    if (o instanceof Request) {
370 		Request other = (Request) o;
371 		return (codeSource == null
372 			? other.codeSource == null
373 			: equals(codeSource.getLocation(),
374 				 other.codeSource.getLocation()))
375 		    && equals(principals, other.principals)
376 		    && equals(perms, other.perms);
377 	    } else {
378 		return false; 
379 	    }
380 	}
381 
382 	private static boolean equals(Object x, Object y) {
383 	    return (x == null) ? y == null : x.equals(y);
384 	}
385 
386 	private static boolean equals(Object[] x, Object[] y) {
387 	    if (x == null) {
388 		return y == null;
389 	    } else if (y == null) {
390 		return false;
391 	    } else {
392 		return Arrays.equals(x, y);
393 	    }
394 	}
395 
396 	public int hashCode() {
397 	    return hash(codeSource == null ? null : codeSource.getLocation())
398 		^ hash(certs) ^ hash(principals) ^ hash(perms);
399 	}
400 
401 	private static int hash(Object obj) {
402 	    return (obj == null) ? 0 : obj.hashCode();
403 	}
404 
405 	private static int hash(Object[] array) {
406 	    int result = 0;
407 	    if (array != null) {
408 		for (int i = array.length; --i >= 0; ) {
409 		    result ^= hash(array[i]);
410 		}
411 	    }
412 	    return result;
413 	}
414 
415 	public String toString() {
416 	    StringBuffer buf = new StringBuffer();
417 	    buf.append("grant\n");
418 	    if (codeSource == null) {
419 		buf.append("    /* bootstrap codebase */\n");
420 	    } else {
421 		URL location = codeSource.getLocation();
422 		if (location != null) {
423 		    buf.append("    codeBase ");
424 		    buf.append(quoteString(location.toString()));
425 		    buf.append('\n');
426 		}
427 		if (certs != null) {
428 		    for (int i = 0; i < certs.length; i++) {
429 			buf.append("    signedby ");
430 			buf.append(quoteString(getCertName(certs[i])));
431 			buf.append('\n');
432 		    }
433 		}
434 	    }
435 	    if (principals != null) {
436 		for (int i = 0; i < principals.length; i++) {
437 		    buf.append("    principal ");
438 		    buf.append(principals[i].getClass().getName());
439 		    buf.append(' ');
440 		    buf.append(quoteString(principals[i].getName()));
441 		    buf.append('\n');
442 		}
443 	    }
444 	    buf.append("{\n");
445 	    for (int i = 0; i < perms.length; i++) {
446 		Permission perm = perms[i];
447 		buf.append("    permission ");
448 		buf.append(perm.getClass().getName());
449 		buf.append('\n');
450 		buf.append("        ");
451 		buf.append(quoteString(perm.getName()));
452 		String actions = perm.getActions();
453 		if (actions != null && actions.length() != 0) {
454 		    buf.append(",\n");
455 		    buf.append("        ");
456 		    buf.append(quoteString(perm.getActions()));
457 		}
458 		buf.append(";\n");
459 	    }
460 	    buf.append("};");
461 	    return buf.toString();
462 	}
